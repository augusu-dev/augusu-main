# Augusu - Minecraft風ゲーム

Three.jsで作られたブラウザベースのマインクラフト風3Dボクセルゲームです。

## 機能

- 🎮 3Dボクセルワールド
- 🏃 プレイヤーコントロール（移動、ジャンプ、視点操作）
- 🧱 ブロックの配置と破壊
- 🌍 自動生成される地形（丘、木）
- 🎨 5種類のブロックタイプ（草、土、石、木、砂）
- ☀️ リアルタイムライティングと影

## プレイ方法

1. プロジェクトをクローン：
```bash
git clone <repository-url>
cd augusu
```

2. 依存関係をインストール：
```bash
npm install
```

3. 開発サーバーを起動：
```bash
npm start
```

4. ブラウザで `http://localhost:8080` を開く

## 操作方法

- **W, A, S, D** - 移動
- **スペース** - ジャンプ
- **マウス移動** - 視点操作
- **左クリック** - ブロック破壊
- **右クリック** - ブロック配置
- **1, 2, 3, 4, 5** - ブロックタイプ選択

## ブロックタイプ

1. 🟢 草ブロック
2. 🟤 土ブロック
3. ⚫ 石ブロック
4. 🟫 木ブロック
5. 🟡 砂ブロック

## 技術スタック

- **Three.js** - 3Dレンダリング
- **JavaScript (ES6 Modules)** - ゲームロジック
